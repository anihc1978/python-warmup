# python-warmup

IBM Python for Data Science, AI & Development — Warm-up Scripts

Practice scripts covering core modern Python features required for data-engineering and analytics work.

## Scripts

| # | File | Topics |
|---|------|--------|
| 01 | `01_list_comprehensions.py` | List, dict & set comprehensions; generator expressions |
| 02 | `02_type_hints.py` | PEP 484/526/604: Optional, Union, Tuple, Callable, Generics |
| 03 | `03_dataclasses.py` | `@dataclass`, `field()`, ordering, `frozen=True`, `asdict()` |
| 04 | `04_pydantic_models.py` | Pydantic v2 `BaseModel`, validators, `ValidationError`, enums |
| 05 | `05_generators.py` | `yield`, infinite generators, custom iterators, `itertools` |
| 06 | `06_higher_order_functions.py` | `map/filter/reduce`, `functools`, `lru_cache`, `partial`, composition |
| 07 | `07_classes_and_protocols.py` | OOP, ABCs, `Protocol`, dunder methods |
| 08 | `08_error_handling.py` | Custom exceptions, try/except/else/finally, context managers |
| 09 | `09_decorators.py` | Function, class-based & parameterised decorators; stacking |

## Requirements

```
python >= 3.10
pydantic >= 2.0
```
